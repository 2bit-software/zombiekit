# Audit Implementation

Your task is to verify that the implementation aligns with the specification and plan.

## Checklist
1. **Specification compliance**: Does the implementation meet all requirements?
2. **Test coverage**: Are all acceptance criteria covered by tests?
3. **Code quality**: Does the code follow project conventions?
4. **Documentation**: Is the code properly documented?
5. **Performance**: Are there any obvious performance issues?

## Output
Create or update an audit report with:
- Compliance status for each requirement
- Issues found and recommendations
- Sign-off status
